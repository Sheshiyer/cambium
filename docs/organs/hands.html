🛠️ hands + conductor — build on-brand

The build organ. It takes a taste brief and a spec-kit tasks.md, resolves every task to a real dev cluster out of 40 hub-and-spoke clusters, injects the brand's taste, builds, and ships each artifact through a fail-closed battery. The hands do the work; the conductor decides which hands and refuses to ship the broken or off-brand.

Sheshiyer/skill-clusters hands / the conductor taste-brief → artifact ○ free

1Overview

The hands are 40 hub-and-spoke clusters / 720 skills — frontend, python, rust, devops, data, ai/ml and 34 more — each a domain capability wing with one <cluster>-orchestrator hub that routes to its spokes on demand. Wrapped around them is a conductor loop that turns a spec-kit tasks.md into a dispatch plan (which cluster builds which task), injects the on-brand taste brief, builds, and then runs a ship-battery gate before anything leaves the bench. Resolution and gating cost nothing; the LLM that actually drives a cluster's orchestrator is the gated extension.

2Internal architecture — the self-similar Φ

input contracts in · the conductor resolves & gates · the clusters do the work · one artifact out.

4Data flow + contract

taste-brief + tasks.md → resolve-task (TF-IDF over 720 skills → cluster) → conduct-execute (dispatch <cluster>-orchestrator, taste-injected) → ship-battery (fail-closed) → artifact

The contract is taste-brief → artifact. resolve-task tokenizes each task and scores clusters by IDF-weighted overlap (a distinctive token like rust or flutter beats a common one), so a task lands on the one cluster whose skills actually cover it — phantom clusters are rejected. conduct-execute then injects the brand's taste brief and dispatches that cluster's <cluster>-orchestrator, which routes to its spokes on demand. Nothing ships until ship-battery passes.

5Homeostasis role

The hands are the build. Where the upstream organs decide the brand's contraction — the taste the artifact must obey — the hands execute that contraction: they resolve each task to the cluster that can honor it and dispatch it taste-injected, so the work is on-brand by construction rather than by review. ship-battery is the fail-closed gate: it runs every check it can find and exits non-zero if any required gate fails, which means no expansive or broken artifact ships. A red battery escalates to the human instead of auto-merging. That is the homeostatic guarantee — the build either converges on the brief or it stops.
