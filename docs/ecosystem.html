How it flows

Build-time — the Cambium pipeline (Φ)
idea → genesis (brandmint: brand-DNA) → taste (skill-clusters: on-brand brief, NIM cortex) → build (hands) → ops (snow-gloves: GTM via Explee). The cortex (NIM + Vectorize) is read/written at every stage; drift → why-handler → reroll or absorb.
Run-time — a shopper try-on (FitCheck 2D)
shopper photo → Shopify widget → signed upload → HDILINT backend → FASHN render → garment-on-body result (consent-gated, retention-bounded). 3D bridal swaps FASHN→Gemini render + R2 storage.
Spend & safety. Every paid service (FASHN, Gemini, NIM, fal/OpenAI/Replicate, Explee) sits behind the conductor's fail-closed gate — no paid organ spawns without explicit per-stage approval. Configs above are env-var names only; secret values live in gitignored .env files and the cortex Worker secrets, never in any repo.
