# Prompt: Component Glyph State Board

Create a high-fidelity Cambium TG mini app component board, landscape 1536x1024.

Use the attached references as strict source material:
- Cambium design system sheet for tokens, glyphs, states, controls, orbit rings, panels, and legends.
- Cambium atlas for glyph personality, rails, packet dots, and contour-field texture.
- Existing Mission Control mobile reference for phone-scale hierarchy and product-language labels.

Goal:
Show the modular component system that will be converted into implementation components for the Telegram mini app. This should look like a practical design-system board for engineers, not a marketing poster.

Sections to include:
1. "Glyphs"
   - Genesis star, Taste capsule, Build triangle, Ops folded slab, Cortex radial ring, Arc crescent, Proof dotted receipt, Gate warning aperture.
2. "States"
   - idle, active, selected, complete, blocked, locked, stale, reduced motion.
3. "Orbit Progress"
   - 0%, 25%, 50%, 75%, 100%, blocked, stale.
4. "Mission Components"
   - BranchArcChip, MissionCard, QuestlineTimeline, ProofList, KpiPulse, GateActionRow.
5. "Motion Primitives"
   - orbitSweep, packetDrift, glyphBreathe, warningAttention shown as 3-frame strips.

Visual rules:
- Deep teal Cambium background, chartreuse active accents, soft mint text, peach warnings, low-opacity contour lines.
- Compact engineering board layout with thin separators and readable labels.
- Use 8px-radius or smaller component corners where possible.
- Include small annotations, but keep them sparse.
- Text should be readable and exact where possible.

Do not include:
- Generic SaaS icons, dashboards, hero sections, stock UI, robots, spaceships, glossy AI gradients, or extra logos.
- Raw product architecture phrases like scene provenance, ecosystem target, R3F, operator map, schema, envelope, quest-ledger, paperclipCommandsData.

Output should communicate: these glyphs, states, circles, highlighted states, rails, and animation primitives are now the modular design vocabulary for Mission Control.
