# Shared Style Spine

All screenshots are concept screenshots for a future React Three Fiber / Three.js Cambium experience. They are not literal browser screenshots, not final UI code, and not a technical scene spec.

Use the accepted Cambium R3F moodboard as the primary visual reference. Preserve its brand-native direction: dark cartographic UI, shallow 2.5D planes, flat zoom panels, abstract organ glyphs, hyphae-like rails, ring orbits, dithered packet particles, and compact stage telemetry.

Cambium brand essence:
- A company compiler: Genesis -> Taste -> Build -> Ops, with Cortex feeding every stage.
- A thin living growth layer where the business grows and remembers.
- Venture-operator, bioluminescent calm, fluid motion, organic-precise type.
- Modular islands that can run independently but still read as one organism.

Visual rules:
- Use abstract R3F-renderable primitives: extruded glyphs, shallow slabs, instanced packets, curves, rings, rails, ribbons, contour fields, labels, and simple material studies.
- Do not use houses, towers, office buildings, city streets, robots, spaceships, literal terrain mountains, or photoreal architecture.
- No neon outer glow, no generic AI purple/blue gradient, no pure black, no noisy marketing hero page.
- Palette: #00272B, #012F34, #E0FF4F, #D6FFF6, #231651, off-cream, tiny peach/warning accent.
- Typography: condensed sans for headings, tiny mono for telemetry, readable labels, restrained editorial density.
- Every frame should make the user's current location in the process visible through a highlighted node, path progress, ring halo, or selected rail.

Output:
- 1536x1024 landscape.
- Polished, high-end product-design screenshot.
- Keep labels sparse and exact where requested.
